## Done channel with struct

Структура Worker с двумя каналами: один для сигнала прекращения работы извне, другой для завершения работы самого воркера. В конструкторе воркер по тикеру выполняет какую-то работу, при получении сигнала о завершении выполняет Graceful shutdow (дожидается выполнения последнего запущенного по тикеру процесса и закрывает свой канал)
